1. Tecnologías de Web APIs

Las APIs son mecanismos que permiten a dos componentes de software comunicarse entre sí mediante un conjunto de definiciones y protocolos . Actúan como una capa intermedia que procesa transferencias de datos , permitiendo abrir funcionalidades a desarrolladores externos y socios .

3. Desarrollo de una Web API

Se deben seguir estándares de la industria como REST y formatos JSON/XML . Definir endpoints claros es vital para la interacción .

Implementar autenticación y autorización es crucial . Herramientas como Swagger o Postman facilitan que otros entiendan cómo usarla .

Protocolo de Seguridad JWT

JSON Web Token (JWT) es el estándar (RFC 7519) para transmitir información segura entre partes .

Estructura: header.payload.signature .

  • Header: Algoritmo y tipo de token .
  • Payload: Información del usuario (Claims) .
  • Signature: Garantiza que el token no fue alterado .
